# Output Guide — Football Salary Intelligence

Field guide to every file produced by the pipeline (`data/results/`).

---

## 01 · Model Coefficients

`data/results/01_model_coefficients/01_linear_model_coefficients/`

| File | Contents |
|---|---|
| `coef_defenders.csv` | OLS coefficients, std errors, t-stats, p-values — Defenders model |
| `coef_midfielders.csv` | Same — Midfielders |
| `coef_forwards.csv` | Same — Forwards |
| `coef_goalkeepers.csv` | Same — Goalkeepers |
| `coef_all_models.csv` | All four models stacked, with a `position_group` column |

Each row is one predictor. Columns: `variable`, `coef`, `std_err`, `t_stat`, `p_value`, `ci_lower`, `ci_upper`.

> Standard errors are **club-clustered** throughout (not HC3).

---

## 02 · Efficiency Scores

`data/results/02_efficiency_scores/`

| File | Contents |
|---|---|
| `efficiency_scores_all.csv` | Full dataset with `efficiency_score`, `predicted_log_wage`, `price_tier` |
| (also served as) `docs/data/efficiency.json` | Same data as JSON for the dashboard |

Key columns:

| Column | Description |
|---|---|
| `efficiency_score` | `log_wage_observed − log_wage_predicted` (residual from OLS model) |
| `price_tier` | `Overpaid` / `Fairly Priced` / `Underpaid` (threshold: ±0.5σ of residual distribution) |
| `Annual_Wages_EUR` | Observed annual wage in EUR |
| `predicted_log_wage` | Model-fitted log wage |
| `position_group` | `Defenders` / `Midfielders` / `Forwards` / `Goalkeepers` |

## 03 · Hypothesis Tests

`data/results/03_hypothesis_tests/`

| Sub-folder | Contents |
|---|---|
| `H1_age_curve/` | Age² peak estimates by position, superstar convexity test results |
| `H2_league_structure/` | Levene's test results per position (W-statistic, p-value) |
| `H3_club_identity/` | ICC estimates per position (FWD=0.187, GK=0.203) |
| `H4_wage_rigidity/` | Interaction regression output (β̂=0.554, t=17.32, p<10⁻⁶⁴) |

---

## 04 · Robustness

`data/results/04_robustness/`

| Sub-folder | Contents |
|---|---|
| `01_bootstrap/` | `ranking_bootstrap_uncertainty_final.csv` — 800-resample bootstrap: mean rank, P10/P90, appear rate per player |
| `02_chow/` | Chow structural-break F-stats and p-values per position × league pair |
| `03_oos_validation/` | Out-of-sample R² by holdout scheme (season-based and league-based) |

**Bootstrap CSV columns:** `Player`, `Team`, `Season`, `side` (Overpaid/Underpaid), `mean_rank`, `p10_rank`, `p90_rank`, `appear_rate`.
